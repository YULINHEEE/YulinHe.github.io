My Projects

AI Sales Co-Pilot Backend Development (2025)

Tools: n8n, Supabase (PostgreSQL), Salesforce SOQL, Pinecone, Azure OpenAI, ElevenLabs
Skills: Multi-agent workflow design, RAG pipeline, SQL & CRM integration, LLM orchestration, API engineering, system deployment

Worked as the lead developer in an industry AI project to build the backend for an AI Sales Co-Pilot used by a major pharmaceutical company. Designed and implemented a multi-agent workflow capable of routing queries across SQL, Salesforce, RAG retrieval, and summarisation agents. Built the system using n8n, integrating Supabase (PostgreSQL + RLS), Salesforce SOQL, Pinecone vector search, and LLM-based reasoning. Delivered a fully functional backend with secure authentication, automated data retrieval, grounding via embeddings, and real-time voice interaction through ElevenLabs.

NLP Job Advertisement Classification & Search Web Application (2024)

Tools: Python, Jupyter, Flask, NLTK, Scikit-learn, Word2Vec
Skills: text preprocessing pipeline, logistic regression classifier, feature engineering

Built an NLP system to automate job-ad categorisation. Processed 776 job ads with regex tokenisation, lemmatisation, stopword filtering, and DF/TF pruning. Engineered CountVectorizer, TF-IDF, and Word2Vec features and trained a logistic regression model achieving 84–86% accuracy with 5-fold cross-validation. Delivered the final classifier in a Flask web app with semantic keyword search and a category recommendation tool for employers.

Colon Cancer Histopathology Image Classification (2025)

Tools: Python, TensorFlow/Keras, NumPy, Matplotlib
Skills: CNN modelling, small-data training, data leakage prevention, augmentation & class balancing, model evaluation & optimisation

Developed binary and multiclass CNN models using histology images from 99 patients, ensuring patient-level split to avoid data leakage. Addressed class imbalance with augmentation, class weighting, and oversampling. Improved binary CNN performance F1 from 0.093 to 0.424 and recall from 0.052 to 0.376 through regularisation, deeper architectures, and hyperparameter tuning. Built a stable multiclass classifier under small-data settings and evaluated models using F1, recall, AUC, confusion matrices, and learning curves.

Supply Chain Delivery Delay Prediction (2024)

Tools: Python, Pandas, NumPy, Scikit-learn, Matplotlib
Skills: feature engineering, route risk modelling, classification, statistical testing, model evaluation

Built a classification model to predict delivery delay risk across shipping routes. Cleaned and encoded supply chain order data, engineered route-level features such as Route Delay Rate and Shipping Mode, and analysed feature significance with Cramér’s V. Trained Logistic Regression and Random Forest models, with Random Forest achieving better AUC and recall. Key predictors included Shipping Mode, Route Delay Rate, and Customer Segment, supported by route-level visualisation to highlight high-risk paths.

TaxGenie: AI Tax Assistant for International Students (2024)

Tools: Python, Sentence Transformers, LLaMA, FAISS vector store, RAG pipeline, Flask
Skills: web scraping, data cleaning, document chunking, embeddings, RAG, LLM integration, prompt design, accuracy validation

TaxGenie is an AI-powered assistant designed to help international students navigate Australian tax filing. Our team collected verified information from the ATO and Australian Government websites, covering residency rules, income types, deductions, and visa-related tax requirements. These documents were chunked, embedded using Sentence Transformers, and stored as vectors for retrieval. When a user asks a question, TaxGenie retrieves the most relevant sources and generates an answer using LLaMA 3.1 70B, ensuring transparency by returning the exact source URL. This project demonstrates end-to-end AI application development, including data sourcing, RAG pipeline design, embedding models, LLM integration, and accuracy assurance through grounded responses.
