Logistics Delay Prediction with Machine Learning

Tools: Python, SQL, Pandas, Scikit-learn, Jupyter
Skills: Machine Learning, Feature Engineering, Classification Modeling, Exploratory Data Analysis

Conducted an in-depth analysis of supply chain delivery delays using Python. Identified key risk factors such as product type and destination region. Compared model performance and interpreted feature importance to support business decision-making.

Job Advertisement Classification with NLP
